# 算术运算符
|运算符|说明|实例|结果|
|-|-|-|-|
|+|加|12.45 + 15|27.45|
|-|减|4.56 - 0.26|4.3|
|*|乘|5 * 3.6|18.0|
|/|除|7/2|3.5|
|//|整除|7/2|3|
|%|取余|7%2|1|
|\**|幂运算/次方运算，即返回x的y次方|2**4|16|

## 加法运算
### 正常的加法运算
```python
m = 10
n = 123
sum = m + n

x = 123.456
y = 789.123
sum2 = x + y
print("sum1 = %d, sum2 = %.2f" %(sum1,sum2))
``` 
运行结果：
```
sum1=133, sum2 = 912.579
```
